# Multi-workshop Operations

## Reset mezi workshopy

Dashboard teď podporuje reset instance z admin panelu.

Doporučený postup:

1. otevři `/admin`
2. vyber konkrétní workshop template
3. spusť reset instance
4. ověř, že:
   - agenda je na začátku dne
   - týmy jsou prázdné
   - sprint checkpoint feed je prázdný
   - challenge completion je vynulované
   - rotace je znovu zamčená
