# Materials

This directory is reserved for participant-facing or print-ready workshop assets that belong to the public workshop product.

Examples that belong here:
- printable reference sheets
- QR cards
- participant handouts
- room-safe visual assets tied to the workshop experience

Examples that do **not** belong here:
- travel logistics
- facilitator packing lists
- deploy checklists
- live event operations
- private follow-up or internal coordination notes

Those operational artifacts belong in the private workshop-instance layer, not in this public template repo.
