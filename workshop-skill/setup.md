# Workshop Setup

## Nejrychlejší cesta

- macOS/Linux: Codex CLI
- Windows nebo macOS: Codex App
- Když se seknete: web fallback nebo pairing s někým, komu setup běží

## Codex CLI

1. Ověřte, že máte přístup ke Codex účtu.
2. Nainstalujte CLI podle interního setup flow vaší organizace.
3. Přihlaste se.
4. Ověřte, že umíte otevřít repozitář a poslat první smysluplný prompt.

## Codex App

1. Nainstalujte aplikaci.
2. Přihlaste se stejným účtem.
3. Otevřete workshop repozitář nebo svůj týmový projekt.
4. Ověřte, že umíte poslat první prompt.

## Když něco nefunguje

- Neztrácejte 20 minut sami.
- Po 7 minutách blokace přejděte na App/Web fallback nebo pairing.
- Pokud nefunguje autentizace, pokračujte s někým ze stolu a vraťte se k setupu později.
