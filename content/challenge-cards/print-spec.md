# Challenge Cards Print Spec

## Formát

- velikost: A6 nebo čtvrtina A4
- oboustranně není potřeba
- velký nadpis, 1 krátký popis, 1 ikonický štítek kategorie

## Barevné rozlišení

- Context Engineering: písková / žlutá
- Workflow: modrozelená
- Advanced: cihlová
- Meta: šedá

## Layout

1. nahoře název karty
2. pod ním kategorie
3. 2-3 věty zadání
4. dole malý štítek:
   - `Před obědem`
   - `Po rotaci`
   - `Kdykoliv`

## Tisková poznámka

Pokud se nestihne tisk, dashboard + `/workshop challenges` jsou primární kanál. Tisk je bonus, ne závislost workshopu.
