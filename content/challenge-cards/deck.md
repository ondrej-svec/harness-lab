# Challenge Cards

## Context Engineering

- `Vytvořte AGENTS.md` — zapište cíle, build/test příkazy a trvalá pravidla.
- `Přidejte build/test příkazy` — agent musí umět svou práci sám ověřit.
- `Napište skill pro code review` — formalizujte jednu opakovatelnou review rutinu.

## Workflow

- `Použijte /plan před kódováním` — napište, z jakého plánu tým vycházel.
- `Rozdělte práci do více vláken` — dvě nezávislé větve práce, jeden integrátor.
- `Delegujte úkol a vraťte se ke kontrole za 10 minut` — nezačínejte mikromanagementem.

## Advanced

- `Spusťte 2 paralelní Codex sessions` — rozdělte problém na dvě nezávislé části.
- `Vytvořte runbook pro deployment` — i když deploy jen simulujete.
- `Napište AGENTS.md pro podsložku` — ukažte, že kontext může být lokální i globální.

## Meta

- `Přesuňte trvalé pravidlo z promptu do AGENTS.md`.
- `Přidejte sekci Done When ke každému tasku`.
- `Napište README pro tým po rotaci, ne pro sebe`.

## Facilitační pravidlo

- Před obědem má každý tým splnit alespoň jednu kartu z `Context Engineering`.
- Po rotaci má každý tým splnit alespoň jednu kartu z `Workflow`.
- Ostatní karty jsou dobrovolné, ale doporučené jako stretch cíle.
