# Plans Archive

Completed and superseded plans live here. Append-only — nothing is ever deleted.

See [`docs/plan-lifecycle-standard.md`](../../plan-lifecycle-standard.md) for when a plan moves into this directory, how inbound links are updated, and the `workshop analyze` checks that keep the parent `docs/plans/` directory containing only live work.

Every plan here retains full git history — use `git log --follow docs/plans/archive/<filename>` to trace it.
